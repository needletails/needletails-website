降低伺服器成本

在生產環境執行後端時，執行時的選擇直接影響計算支出。伺服端 Swift（Vapor、Hummingbird、SwiftNIO）通常比許多流行執行時占用更少記憶體、單核處理更多請求。您可以用更小或更少的實例承載相同流量——雲端帳單下降且不犧牲可靠性。

一年下來，選擇更高效的執行時可在相同負載下將伺服器帳單減少數百至數千美元。下表為單台小型雲端實例上約每月 1–2 百萬請求的簡單 JSON API 的合理參考。

記憶體使用決定所需伺服器規模。流量能力是一台伺服器可處理的請求數。更低記憶體 + 更高能力通常意味相同流量下月費更低。
